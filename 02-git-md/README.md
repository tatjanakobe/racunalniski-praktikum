## Bližnjice na tipkovnici

* _Kopiraj_ označeno v odložišče: Ctrl+C (**C**opy)
* _Izreži_ označeno v odložišče: Ctrl+X 
* _Prilepi_ vsebino odložišča: Ctrl+V
* ctrl n odpre novo datoteko
* ctrl a označi vse 
* ctrl preskakuje besede
* shift označi vse
* ctrl s shrani datoteko
* ctrl w zapre
* ctrl shift t
* ctrl o odpre datoteko
* ctrl d izbere eno besedo
* ctrl+shift+p
* ctrl a oznaci use
* shift+insert v gitbashu pastaš 




<!-- 2. nivojski razdelek -->
## Izvorna koda

Včasih pride prav značka `kbd` za tipke. Značko uporabimo takole
```
<kbd>Ctrl</kbd>
```
